# Student Registration System

A simple web application to manage student registrations. Users can add, edit, and delete student records. All data is stored in the browser's localStorage.

## Features

- Add new student records (Name, ID, Email, Mobile Number)
- Edit existing student details
- Delete student records
- Form validation (name must be text, mobile number must be 10 digits)
- Data stored using localStorage
- Responsive and user-friendly interface

#Technologies

- HTML
- CSS
- JavaScript

## How to Use

1. **Clone the repository:**
   git clone https://github.com/dilnakt-bnglr/student-registration-system.git
2. **Open the project folder:**
   cd student-registration-system
3. **Open `index.html` in your browser.**
